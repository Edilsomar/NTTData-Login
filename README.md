# 🔐 Login NTT Data

📄 Projeto de uma tela de login moderna com design inspirado em padrões corporativos.

---

## 🚀 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar **HTML, CSS e JavaScript**, focando na criação de interfaces modernas e funcionais.

A proposta foi recriar uma tela de login com aparência profissional, aplicando conceitos de **design, responsividade e validação de formulário**.

---

## 🧠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Boxicons (ícones)

---

## 🎯 Funcionalidades

* ✅ Campo de email e senha
* ✅ Validação de campos obrigatórios
* ✅ Feedback de erro e sucesso
* ✅ Interface responsiva
* ✅ Opção "Remember password" (visual)

---

## 🎨 Destaques do projeto

* Interface moderna com efeito de transparência e gradiente
* Uso de ícones para melhorar a experiência do usuário
* Layout adaptável para diferentes tamanhos de tela
* Estrutura de código organizada
